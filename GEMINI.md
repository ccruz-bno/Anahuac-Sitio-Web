# GEMINI.md

Adaptador para Google Antigravity / Gemini. Las instrucciones reales del proyecto viven en un archivo neutral:

**👉 Lee y sigue [`AGENTS.md`](./AGENTS.md) como reglas always-on.**

Antigravity lee `AGENTS.md` de forma nativa (cross-tool). Este archivo existe solo para que, si Antigravity
prioriza `GEMINI.md`, la regla siga apuntando a la fuente única. No dupliques instrucciones aquí.

Recordatorio del gate (ver `docs/readiness.md`): no construyas sin acceso a Figma (MCP/skill) e ingesta inicial.
El Figma MCP se instala en Antigravity vía ⋯ → MCP servers; el nombre de la tool no importa, solo la capacidad.
