# Adapter: Astro

Para sitios estáticos o prototipo. Convenciones: `docs/platforms/astro.md`.

```
astro/
├── src/styles/tokens.css
├── src/components/<X>.astro
├── src/layouts/ · src/pages/
└── astro.config.mjs
```

> Astro no da autoadministración nativa; para edición por contenido usar `hubspot` o `wordpress`.

## Estado
Se activa cuando `project.json → platform = "astro"`.
